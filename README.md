# TFT 4 Tộc Khu Vực — Lõi XP Advisor

App gợi ý kích hoạt **Lõi XP 4 Tộc Khu Vực** nhanh nhất trong TFT Set 16 (DTCL Mùa 16).

## Tính năng

- Chọn tướng đang có trên bàn
- Chọn Ấn tộc hệ đang có
- Tự động tính combo tướng rẻ nhất để đủ 4 tộc Khu Vực
- Hiển thị top 6 combo theo tổng giá vàng
- Filter theo tộc, tìm kiếm tên tướng

## Cách dùng

1. **Chọn Ấn** (nếu có) — mỗi ấn tính là 1 tộc free
2. **Chọn tướng** đang trên bàn
3. Xem gợi ý combo mua thêm để đạt 4 tộc

## Deploy

Tự động deploy lên GitHub Pages qua GitHub Actions mỗi khi push lên `main`.

**Live:** https://degokisss.github.io/tft-4-trails/

## Stack

- React 18 (CDN)
- Vanilla CSS
- GitHub Pages + GitHub Actions
